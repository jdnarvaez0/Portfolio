📸 Cómo guardar la imagen OG

  1. Usa Chrome/Edge y abre DevTools (F12)
  2. Presiona Ctrl+Shift+P y escribe "Capture screenshot"
  3. Se guardará como PNG de 1200x630px
  4. Guárdalo como public/og-image.png

O: Usa Screenshot Machine con URL de este archivo
